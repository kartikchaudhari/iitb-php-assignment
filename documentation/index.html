A) Getting Started - top

Extact the zip file to htdocs directory using any zip extraction program (Ex. 7zip).



Directory Structure

coordinator-portal: it is seperate web application directory. contains all the neccessary css, javascript, images and php files required for Institute Workshop Coordinator Portal.

student-portal: it is seperate web application directory. contains all the neccessary css, javascript, images and php files required for Student Feedback Portal.

database: it contains the database .sql file, required to setup database for both web applications.

B) Database Setup - top

Following are the for Database Setup.

  1. open phpmyadmin (if we are using XAMPP/LAMP follow this url http://localhost/phpmyadmin/index.php.
  2. Click on databases tab, enter database name and press Create button.

  3. Click on recently created database from database list (shown in above step).
  4. You will redirected to database operation page, from operation strip, click on Import tab.

  5. It will open file import wizard.
  6. Now navigate to extracted zip file, in that open database directory.
  7. you will find a database SQL file, we need to create zip file of it.
  8. Now returned to file import wizard, browse that zip file and press Go button.
  9. After sucessfully import, click on Structure tab from operation strip.
  10. We will have list of tables as shown in below figure.

Database is setup successfully.

C) Configuration - top

Both the web application have their seperate configuration files, so if we install them on different domain, they will work as expected.

While both the web application are dependent on each other, they use same and common database.

Following are the Steps to configure each web applicaion:

Configuration of Institute Workshop Coordinator Portal
  1. Navigate to coodinator-portal/include directory.
  2. From list of file edit config.php file using any text editor.

    Variable Description
    $user : Stores the username of database user.
    $password : Stores the password of database user.
    $host : Stores the hostname of database server.
    $db : Stores the name of database (database created in Database Setup step.)
    $base_url : Stores the the absoule URL, where php, html, css, js and image files of Institute Workshop Coordinator Portal are stored.

    Note: These variables are very important to run web application smoothly.


Configuration of Student Feedback Portal
  1. Navigate to coodinator-portal/include directory.
  2. From list of file edit config.php file using any text editor.

    Variable Description
    $user : Stores the username of database user.
    $password : Stores the password of database user.
    $host : Stores the hostname of database server.
    $db : Stores the name of database (database created in Database Setup step.)
    $base_url : Stores the the absoule URL, where php, html, css, js and image files of Student Feedback Portal are stored.

    Note: These variables are very important to run web application smoothly.

Both the web application are configured successfully.
